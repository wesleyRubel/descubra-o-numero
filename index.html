Jogo do descubra o número

Um pequeno jogo feito em JavaScript onde o jogador tenta adivinhar um número gerado aleatóriamente pelo navegador. Por meio de uma interface feita com HTML e CSS o usuário escolhe um número, e ao apertar o botão "Apostar" o script ira receber o número informado pelo usuário e verificar se é igual ao número gerado pelo script.

ERROS: 0

CHANCES: 6

Dica: É um número entre 1 e 100:
